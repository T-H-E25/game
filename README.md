## Installation

Clone the repository and install dependencies:

```bash
git clone https://github.com/T-H-E25/game.git
cd game
npm install
```

## Usage

To run the project in development mode:

```bash
npm run dev
```

To build the project for production:

```bash
npm run build
```

To preview the production build:

```bash
npm run preview
```
